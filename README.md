# 🎵 lx-blive 

一个为 Bilibili 直播间打造的弹幕点歌机器人内核。支持集成 **洛雪音乐 (LX Music)** 与 **MusicFree** 双生态插件引擎，支持跨平台全网搜歌与播放。

## ✨ 核心特性（它居然还能实现这些功能）

尽管代码很烂，但它确实塞进去了很多功能：

- 🎮 **B站直播间弹幕点歌**：监听 `!点歌 歌名` 弹幕。
- 👥 **精细的权限控制**：支持分别设置 主播、房管、舰队(大航海)、普通用户的点歌上限与 CD 冷却时间。
- 🔌 **双插件引擎支持**：
  - **LX 生态**：支持直接导入洛雪音乐（LX Music）的 `.js` 音源脚本。
  - **MF 生态**：支持导入 MusicFree 的 `.js` 插件脚本。
- 🔍 **多平台聚合搜索**：内置 tx(企鹅)、wy(网易)、kg(酷狗)、kw(酷我)、mg(咪咕) 等搜索解析映射。
- ⚙️ **四大播放策略（引擎模式）**：
  1. **模式 1 (传统被动)**：被动调用 LX 协议唤起本地播放器。
  2. **模式 2 (半主动推送)**：SDK 搜索后精准推送给洛雪客户端播放。
  3. **模式 3 (全主动 LX)**：完全脱离客户端，由后台导入的 LX 插件直接解析并使用内置播放器播放。
  4. **模式 4 (全主动 MF)**：完全脱离客户端，由后台导入的 MusicFree 插件跨源并发搜索并解析播放。

## 🚀 运行与使用

### 环境要求
- [Node.js](https://nodejs.org/) (推荐 v16 以上)
- 一颗对 Bug 极度包容的心

### 安装依赖
```bash
# 克隆仓库后，安装依赖
npm install
```

### 启动项目
```bash
# 启动 Electron 应用
npm start
```

### 基本使用流程
1. 打开软件，点击左侧栏的 **“系统设置”**。
2. 在 **“B站直播间”** 选项卡填入你的 `Room ID`（Cookie 选填）。
3. 在 **“播放模式与插件”** 选项卡中，选择你的播放模式。
   - *强烈建议使用模式 3 或 4，并点击右侧按钮导入你准备好的 `.js` 插件。*
4. 保存设置后，在首页控制台查看是否提示“B站直播间：已连接”。
5. 观众发送弹幕 `!点歌 乌梅子酱` 即可体验。

## 📂 史山导览 (给勇敢者的源码地图)

如果你执意要看源码，这里是主要文件的罪证分布：
- `main.js`: 缝合怪主进程，包含了极其混乱的沙盒创建和 IPC 跨进程调度。
- `core/MusicBot.js`: 核心逻辑，混合了 B站 WS 连接、队列管理和乱七八糟的歌曲数据格式化清洗。
- `lx-*.js`: 各大平台的 API 抓取脚本，包含了各种被硬编码的神秘请求头和加密解密算法。
- `MF-plugin-tester.main.js` & `lx-source-tester.sandbox-preload.js`: 为了强行兼容两种不同生态的插件，用 `Function` 和 `contextBridge` 捏造出来的虚假执行环境。
